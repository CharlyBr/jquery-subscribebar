jQuery subscribe bar example: bootstrap settings

This example is showing the subscribe bar when none of the settings have been altered. It is also using the default stylesheet provided with the subscribe bar.

Code for this example:

            $.subscribeBar({
                emailClass: "form-control",
                formClass: "form-inline",
                msg: "Stay tuned!",
                submitClass: "btn btn-warning",
            });
